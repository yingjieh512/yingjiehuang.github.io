Research Experience

  • 2025 – Present
    Graduate Student Researcher — Neural Network Explainability
    UCLA
    Conducting research on neural network interpretability using the Sufficient Input Subsets (SIS) framework (Carter et al., NeurIPS 2021). SIS identifies the minimal subset of input features sufficient for a black-box model to maintain its high-confidence prediction with all other features masked. Extending the framework along five directions: Probabilistic SIS (per-feature inclusion frequencies via randomized tie-breaking); Hierarchical SIS (multi-scale explanations from pixel to patch to region); Contrastive SIS (feature sets supporting the predicted class while suppressing a foil); SHAP-guided acceleration (reducing the O(n²) query cost of backward selection); and SIS stability as a model robustness metric (Jaccard similarity across perturbed inputs).

Triangular Mesh Generator — Computational Geometry

Implemented a 2D unstructured triangular mesh generator in Julia based on the DistMesh algorithm (Persson & Strang, SIAM Review 2004).
